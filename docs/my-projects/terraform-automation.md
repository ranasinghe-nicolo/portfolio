# Terraform Automation

Progetto orientato alla creazione di infrastruttura come codice con Terraform.

## Obiettivo

Automatizzare il provisioning di risorse cloud e ambienti di test in modo ripetibile e documentato.

## Attività principali

- definizione di moduli Terraform per rete, compute e storage
- gestione dello stato remoto e workflow sicuro
- integrazione con pipeline CI/CD per deploy automatici

## Risultati

- infrastruttura riproducibile in più ambienti
- rollback controllato e gestione di variabili sensibili
